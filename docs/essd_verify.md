## essd verify

Verify signatures in DSSE envelope using specified keys

```
essd verify [flags]
```

### Options

```
  -h, --help              help for verify
  -k, --key stringArray   key to use for verifying signatures (specify sigstore using fulcio:<identity>::<issuer>)
```
