## essd cat

Concatenate specified parts of DSSE envelope

```
essd cat [flags]
```

### Options

```
  -d, --decode-base64   base64 decode payload
  -h, --help            help for cat
  -p, --payload         envelope payload
  -t, --payload-type    envelope's payload type
      --summary         summary of envelope
```
